Importing Existing Accounts

If you're switching to Hyperpass from another password manager, you'll likely want to import your accounts. First, refer to your previous password manager's user guide for instructions on how to export your login data: Google, Firefox, Bitwarden, LastPass. If given an option for the type of export, select the unencrypted option. Your export should result in a .CSV file.

Next, let's go through importing the exported data into Hyperpass. First, log in to the Hyperpass app, navigate to the Options page, and press the 'Import' button. Then, select the format of the vault you want to import. The format you select should correspond with the name of your previous password manager.

In most cases, you will want to leave the 'Trim' option enabled. This will remove unnecessary parts of the URLs your accounts are associated with. Finally, select the 'Mode' you want. 'Merge' is suitable for most cases, while 'Append' is conservative and 'Overwrite' is destructive. If you're curious about the specifics of a mode, select it from the dropdown and its corresponding description will be displayed at the bottom of the page.

When you have selected the import options you want, press the 'Import' button on the top right of the app. This will open a file selection window where you can select the .CSV file that you exported from your previous password manager. Once selected, the account data will be imported according to your settings.
