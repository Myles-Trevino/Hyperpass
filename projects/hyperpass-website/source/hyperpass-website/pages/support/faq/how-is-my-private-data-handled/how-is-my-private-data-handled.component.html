How Is My Private Data Handled?

Due to the sensitive nature of the data Hyperpass handles, this is an important question for any user to ask. The fundamental policy Hyperpass is built on is that none of your sensitive data leaves your computer unless it is securely encrypted, and the key to unlocking it only lies with you. The encrypted version of your vault is stored on Hyperpass servers so that it can be synchronized across your devices. However, because your master password does not leave your device, only you have the ability to unlock it. Your vault will only ever be decrypted on your own devices. This ensures that even if the Hyperpass servers were to be compromised, unwanted parties would not be able to access your sensitive data.

One advantage of Hyperpass over other solutions is that it is open source, meaning that the program code is available for anyone to see. You can check the source code yourself and see excatly what is going on behind the scenes. If you're interested, please visit the Hyperpass GitHub page.
